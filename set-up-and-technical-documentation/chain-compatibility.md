# Chain Compatibility

Tally is currently compatible with the following chains:

* Ethereum
* Arbitrum One
* Arbitrum Nova
* Avalanche
* Base
* Blast
* BNB Smart Chain
* Corn
* Filecoin
* Flow Mainnet
* Gnosis
* HashKey Chain (coming soon)
* Kinto
* Linea
* Lisk
* Moonbeam
* Nova
* Optimism
* Polygon
* Polygon zkEVM
* RARI Chain
* Rootstock Mainnet
* Scroll
* Soneium
* Swellchain
* Vana Mainnet
* Viction
* Zircuit
* ZKsync Era

If you would like to integrate your chain with Tally email us at [hello@tally.xyz](mailto:hello@tally.xyz).
